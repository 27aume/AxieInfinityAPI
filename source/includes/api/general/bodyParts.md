## Body Parts

> Returns JSON structured like this:

```json
[
  ...,
  {
    "partId": "mouth-lam-handsome",
    "class": "aquatic",
    "specialGenes": "mystic",
    "type": "mouth",
    "name": "Lam Handsome"
  },
  ...,
]
```

Allows to retrieve some data of all body parts.

### Endpoint

`GET /body-parts`
